## 🚀 התקנה והרצה

### דרישות מקדימות
- Node.js 18+
- npm או yarn
- חשבון Supabase (חינמי)

### התקנה מקומית

```bash
# שכפל את הפרויקט
git clone https://github.com/Yklein888/family-finance-manager.git
cd family-finance-manager

# התקן תלויות
npm install

# העתק .env.example ל-.env.local
cp .env.example .env.local

# הגדר משתני סביבה ב-.env.local:
# VITE_SUPABASE_URL=your_supabase_url
# VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# הרץ את המערכת
npm run dev
```

### הרצת Migrations

```bash
# אופציה 1: דרך Supabase Dashboard
# 1. https://supabase.com/dashboard/project/YOUR_PROJECT/sql/new
# 2. העתק את תוכן הקבצים מ-supabase/migrations/
# 3. Run

# אופציה 2: דרך CLI
npx supabase db push
```
